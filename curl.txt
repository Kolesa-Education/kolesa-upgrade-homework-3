curl - это утилита, которая позволяет работать с URL страницами и передать файлы.

Возможности:
- curl url_файла (загрузка файла по URL),
- curl -T название_и_путь_до_файла адрес_ftp_сервера(можно передать файл на ftp),
- curl -F параметры_запроса url_сервера (отправить данные через POST запрос),
- curl -c/b файл_куки url_сервера (отправить и принять куки),
- curl -I url_сервера(прием заголовков),
- curl -x url_proxy url_сервера(можно сделать запрос через прокси).

HTTP/2 200: HTTP - протокол, 2 - версия, 200 - код ответа
server: nginx - вебсером является nginx
-I - чтение заголовков
